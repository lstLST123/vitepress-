>
> 学习时示例代码：https://gitee.com/cao-shen-yang/springboot3

**特性：**

- 快速创建独立的 **Spring** 应用
- 内嵌 `Tomcat`、`Jetty`、`Undertow` 等 **Servlet** 容器
- 生态完善，提供各种 **starter**，方便整合
- 自动配置 **Spring** 以及第三方库
- 提供生产级特性：如监控指标，健康检查，外部化配置

**总结：简化开发、简化配置、简化整合、简化部署、简化监控、简化运维**
